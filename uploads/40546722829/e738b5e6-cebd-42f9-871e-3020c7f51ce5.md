# GopherAI 项目文档

## 项目介绍
GopherAI 是一个基于 Go 语言开发的 AI 聊天机器人系统，支持多轮对话、RAG 知识库和图像识别功能。

## 技术栈
- 后端：Go + Gin + GORM
- 前端：Vue 3 + Element Plus
- 数据库：MySQL + Redis
- AI 模型：阿里云通义千问

## 核心功能
1. 用户注册登录（邮箱验证码）
2. AI 多轮对话
3. RAG 知识库（支持上传文档）
4. 图像识别
5. 语音合成（TTS）

## 特殊功能
项目使用了 RedisStack 的向量搜索功能来实现 RAG 检索。

## 项目特点
- 支持多种对话模式
- 集成了 MCP 协议
- 提供了完善的用户管理系统
- 支持文件上传和内容检索
